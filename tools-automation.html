Automation tools compared

n8n

n8n

What it does

An open-source, node-based workflow automation platform (self-hostable or n8n Cloud) with native AI/LangChain nodes for building agents, RAG pipelines, and chatbots.

Best use case

Technical teams or developers building complex, multi-step AI agent workflows that need custom code (JS/Python nodes) mixed with 400+ app integrations.

Limitations

Steeper learning curve than Zapier or Make — built for technical users. Billing is by whole "workflow executions," not per-step, but concurrent-execution caps are low on entry paid tiers.

Free tier

No permanent free cloud plan — the cheapest hosted plan starts at €20/month for 2,500 executions/month. What's genuinely free is the self-hosted Community Edition: unlimited executions, workflows, and users, but you host it yourself with no official support.

Privacy

Cloud-hosted data is stored in Frankfurt, Germany (EU); self-hosted instances keep data wherever you host them. Self-hosted paid tiers send daily telemetry/license-check data, though general telemetry can be disabled.

Zapier

Zapier

What it does

No-code automation ("Zaps") connecting 9,000+ apps, plus Tables, Forms, autonomous Zapier Agents, Chatbots, and a Zapier MCP for connecting AI models to app actions.

Best use case

Non-technical business users automating simple triggers between mainstream SaaS apps — broadest app coverage of the four tools on this page.

Limitations

Free plan is restricted to 2-step Zaps only, with a slower 15-minute polling interval versus 1-2 minutes on paid tiers. AI steps now consume the same shared task pool as everything else, so heavy AI use burns through the free allowance fast.

Free tier (confirmed)

100 tasks/month, unlimited Zap workflows/Tables/Forms, but Zaps limited to 2 steps. Separately: Zapier Agents Free gives 400 "activities"/month, and Chatbots Free allows building up to 2 chatbots using GPT-4o-mini/GPT-3.5.

Privacy

Data is processed/stored in the US and other countries where Zapier operates; complies with EU-U.S. and Swiss-U.S. Data Privacy Framework for cross-border transfers. Uses third-party ad/analytics partners that may collect usage data.

Make

Make (formerly Integromat)

What it does

A visual, node/graph-based automation builder ("scenarios") across 3,000+ apps, with AI agent-building tools, an MCP server, and agentic automation features.

Best use case

Users who want Zapier's ease of use but with more visual branching logic (routers, iterators) for moderately complex multi-branch workflows without writing code.

Limitations

Free plan enforces a 15-minute minimum interval between scenario runs — no near-real-time automation on the free tier. Credit-based pricing means AI-heavy or data-heavy modules burn through credits fast, since each action step costs a credit.

Free tier (confirmed)

1,000 operations/credits per month with no time expiration, full no-code visual builder, 3,000+ apps, routers and filters — but the 15-minute minimum run interval applies.

Privacy

Make's pricing page doesn't detail data-retention specifics; a separate privacy/GDPR notice exists but wasn't independently verified for this comparison — check it directly before relying on any specific retention claim.

Gumloop

Gumloop

What it does

An AI-native, no-code automation builder for building AI agents and workflows with built-in AI nodes — image generation, code sandboxes, hosted agent pages — rather than just app-connector logic.

Best use case

Teams wanting pre-built, AI-first automations (lead qualification, competitor analysis, support agents) rather than generic app-to-app plumbing — Gumloop's own use-case library is AI-task-centric, not integration-centric.

Limitations

Free plan is capped at 1 seat and only 1 active trigger, with very limited concurrency (2 concurrent runs, 5 concurrent agent interactions). AI-heavy nodes consume credits faster than plain data-move steps.

Free tier (confirmed)

5,000 credits/month, 1 seat, 1 active trigger, 2 concurrent workflow runs, 5 concurrent agent interactions, unlimited agents/flows, forum support.

Privacy

The Enterprise tier explicitly offers "Incognito Mode," custom data-retention rules, and AI model access control — implying Free/Pro tiers don't include these controls by default, though this wasn't independently confirmed beyond the pricing page's feature grid.

Anthropic

Cowork

What it does

A mode of Claude Desktop built for non-developers that executes multi-step knowledge work directly on your files — research synthesis, document preparation, file organization — rather than just chatting back. Uses the same agentic architecture as Claude Code.

Best use case

Researchers, analysts, operations and finance teams doing file-heavy, non-technical work: organizing local folders, preparing reports from source documents, extracting data from unstructured files like contracts or receipts.

Limitations

Requires Claude Desktop to stay open with the computer awake — closing the app ends the session. No mobile or web version of full sessions. Memory persists only within a Cowork "project," not across separate sessions. Not yet captured in the Compliance API.

Free tier

None — Cowork is gated to paid plans only (Pro, Max, Team, Enterprise). It also consumes noticeably more usage allocation than standard chat, though Anthropic doesn't publish an exact multiplier.

Privacy

Follows the same account-type privacy policy as the rest of Claude: consumer accounts train only if "Help improve Claude" is on; Team/Enterprise are excluded from training by default. Code/shell commands run in an isolated VM; file/network access is limited to folders and permissions you explicitly grant.
